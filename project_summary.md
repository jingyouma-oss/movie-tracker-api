## Movie Tracker Summary

Movie Tracker is a full stack web application for organizing movies by watch status and saving personal reviews for each title. Users can browse a movie list, add new movies, update their watch progress, and delete titles they no longer want to track. Each movie can also have multiple reviews, which makes the app useful for students, casual movie fans, or anyone who wants a structured personal watchlist. The frontend is built with Next.js, the backend uses FastAPI, and all data persists in PostgreSQL.
