Short description:
Movie Tracker AI Assistant is a full stack web application for managing a personal movie watchlist. It uses a Next.js frontend, a FastAPI backend, and a PostgreSQL database with related movie and review tables. Users can create, view, update, and delete movie records through the UI, and the app also includes AI-powered chat, personalized recommendations, and an agent endpoint that can call tools to manage the movie collection through natural language. This project demonstrates full stack CRUD development, database persistence, AI integration, and agentic tool use in one complete application.
